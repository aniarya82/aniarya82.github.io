# Hub
A git wrapper to extend its features.

## Installation
brew install hub

## Uses
{% highlight css %}
hub clone wurmlab/genevalidator
git clone https://www.github.com/wurmlab/genevalidator
{% endhighlight %}

## Alias
```
hub alias
```
